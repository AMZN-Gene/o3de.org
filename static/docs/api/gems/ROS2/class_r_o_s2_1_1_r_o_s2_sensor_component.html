Detailed Description

Captures common behavior of ROS2 sensor Components. Sensors acquire data from the simulation engine and publish it to ROS2 ecosystem. Derive this Component to implement a new ROS2 sensor. Each sensor Component requires ROS2FrameComponent. For high frequency sensors also derive PhysicsCallbackHandler.
